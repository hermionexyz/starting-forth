# Starting Forth

Learn the programming language Forth using the book "Starting Forth".


https://www.forth.com/starting-forth/2-stack-manipulation-operators-arithmetic/

## Expressions

forth	|	explanation
:-		|	:-
`/mod`	|	`( n1 n2 -- rem quot )`
`mod`	|	`( n1 n2 -- rem )`
`swap`	|	`( n1 n2 -- n2 n1 )`
`dup`	|	`( n -- n n )`
`over`	|	`( n1 n2 -- n1 n2 n1 )`
`rot`	|	`( n1 n2 n3 -- n2 n3 n1 )`
`drop`	|	`( n -- )`
